Este es un sitio diseñado por Jose Antonio Amaya, usando estandadares de las web que permitan la utilización de funcionalidades de ES6 bajo las nuevas caracteristicas de Javascript para comunicación asincrona a través de un servidor, adicional a esto la aplicación se anima con una librería de animación conocida como GSAP de GreenSock y hace un reconocimiento de los textos agregados al input recorre un Array de nombres de pokemon y devuelve la coincidencia almacenada en un objeto con sus poderes e imagen asociada.
